My Notes

  • React JS helps to create virtual DOM then the regular DOM which helps to speed the performance of the app.
  • Is is also used to develop user interfaces.
  • React JS is made up of multiple components which can be reused
  • The latest version of React JS is 16.8
Pre-requites

Javascript - this, filter, map & reduce

ES6 - let & const, Arrow functions, Template literals, Default parameters, Object literals, Rest & Spread operators, Destructuring assignment

components : are a part of user interfaces written in js files in the form of code. There are 2 types of components.
  • Stateless Functional Components

    E.g : function welcome (props) { return Hello, {props.name}; }

  • Statefull Class Components

    E.g : class welcome extends React.Component { render() { return Hello, {this.props.name}; } }
